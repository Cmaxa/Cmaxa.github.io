## C 列出数组中的所有元素：
``` bash
#include<stdio.h>
   int main()
    {
        int i=0;
        for(i=0;i<256;i++)
        {
            printf("[%d]:---192.168.1.%d\n",i,i);
        }
        return 0;
    }
```
