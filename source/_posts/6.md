## C code指针与数组，内存的关系
``` bash
#include <stdio.h>
#include <stdlib.h>

int main()
{
    int a[5]={1,2,3,4,5},*p=a;

    printf("数值：%d\n",*p);
    printf("内存里的地址：%d\n",p);
    return 0;
}

```
